# 🤖 AI Chatbot (GPT-4o-mini)

> Context-aware AI chatbot with conversation memory built with OpenAI's GPT-4o-mini API.

## What it does
- Remembers conversation history across messages
- Uses a custom system prompt to define personality
- Tracks token usage for cost monitoring
- Answers questions accurately using GPT-4o-mini

## Technologies used
- Python 3.11
- OpenAI API (GPT-4o-mini)
- Google Colab

## How to run
1. Clone this repository
2. Install dependencies: pip install openai python-dotenv
3. Add your OpenAI API key as OPENAI_API_KEY
4. Run the notebook in Google Colab

## What I learned
- System prompt engineering
- Multi-turn conversation history management
- API integration and secure key handling
- Token tracking for cost monitoring
